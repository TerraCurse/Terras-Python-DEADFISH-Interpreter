# Terras-Python-DEADFISH-Interpreter
A simple interpreter for DEADFISH.
## COMMANDS
- i ; Increases accumulator.
- d ; Decreases accumulator.
- s ; Squares the accumulator. (acc*acc)
- o ; Prints the current accumulator value as an ASCII character.
- r ; Resets the accumulator back to zero.
## EXAMPLES:
*Hello* <br>
``iiiiiiiisiiiiiiiiodddoiiiiiiiooiiior`` <br>
**Outputs: HELLO**
